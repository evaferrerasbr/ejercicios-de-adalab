Tecnologías web

Las tecnologías web sirven para acceder a los recursos disponibles en Internet o en las intranets utilizando un navegador. En el curso intensivo de Adalab estudiaremos las siguientes:

  • HTML

    HTML es el lenguaje de marcado para la elaboración de páginas web

  • CSS

    Es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en HTML

  • JavaScript

    JavaScript es un lenguaje de programación o de secuencias de comandos que te permite implementar funciones complejas en páginas web
